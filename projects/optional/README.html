Plug-in: org.eclipse.ant.core

For some time there has been a memory leak in the org.eclipse.ant.core plug-in supplied with Eclipse. (The actual problem is with Ant itself). Until the problem is fixed you may want to replace the supplied plug-in with this patched version.

org.eclipse.ant.core_2.1.0 is for Eclipse 2.1.0 and org.eclipse.ant.core_2.1.1 is for Eclipse 2.1.1.
