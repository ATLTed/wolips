Step by Step instructions for PDE developers

Check out WOLips as.. (Not as project)

Check out WOProject as Project

Build WOProject(On the console run ant dist)

Copy cajenne.jar and woproject.jar from the dist/org.objectstyle.wolips.VERSION/lib to the wolips/lip folder checked out previously.

Add a classpathvariable JAVA_WO_FOUNDATION to Eclipse and let it point to JavaWOFoundation.jar

Right click on the wolips project root und select update classpath.
