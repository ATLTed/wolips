Step by Step instructions for PDE developers

Check out WOLips as.. (Not as project)

Check out WOProject as Project

Build WOProject(On the console run ant dist)

Copy cajenne.jar and woproject.jar from the dist/org.objectstyle.wolips.VERSION/lib to the wolips/lib folder checked out previously.

Right click on the wolips project root und select refresh.

Go to Window/Preferences/Plugin Development/Target Platform and choose Select all.

Right click on the wolips project root und select update classpath.(Option is only available in JavaViews.)

Add a classpathvariable JAVA_WO_FOUNDATION to Eclipse and let it point to JavaWOFoundation.jar. Add this variable to .classpath.

To use the WOLips ant tasks right click on the org.objectstyle.wolips.ant package and export the selection as wolips.jar to the wolips/lib folder (only needed during development).

Eclipse uses ant 1.4.1 as default, to use the ant plugin with ant 1.5 you can add the proper jars in the external tools preferences (see "Using a different version of Ant" in the Eclipse help).
