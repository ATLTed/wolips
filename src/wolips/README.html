WOLips requires Eclipse 2.1.

To install WOLips open the update manager perspective and point it to this directory. In the optional directory is a patch for the org.eclipse.ant.core plugin. Copy it in the Eclipse plugins directory. Don't forget to make a backup of you existing org.eclipse.ant.core plugin.

Install help is available online.

Please read the Getting Started section of the WOLips User Guide. You access the Eclipse built-in help by selecting Help > Help Contents from the workbench menu.
