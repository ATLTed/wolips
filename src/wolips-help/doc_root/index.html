WOLips User Guide

WOLips is developed by a small team at a fast rate. The documentation is typically not written by the actual developers - it is written by early adaptors (users like you) who test and document as they learn. Inevitably this means there is a lag between when a new feature is introduced and when it is documented. Add to this that the existing features are constantly refactored and improved upon…

Please read the documentation!

  • Report problems
  • Request additions
  • Submit additions or corrections

Do not hesitate to ask questions or participate in discussions on the WOProject mailing list.
