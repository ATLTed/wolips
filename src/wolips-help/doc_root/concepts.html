Concepts

  • WOLips is a plugin for eclipse that support development of WebObjects applications and frameworks. WOLips is a part of WOProject.
  • eclipse is a kind of universal tool platform - an open extensible IDE for anything and nothing in particular.
  • WOProject is a set of OS and IDE independent tools (based on Ant) for working with WebObjects.
  • Ant is a Java-based build tool.

The concepts of WOLips are the those of eclipse, WebObjects, WOProject and Ant.
