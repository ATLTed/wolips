Importing projects with localized components doesn't work. ?
No imgages in the Eclipse WOLips help. ?

When you start eclipse for the first time with WOLips installed a file ~\Library\wobuild.properties should be created in your user home directory. Make sure this actually did happen!

If not you’ll have to create it yourself. It should contain this:

#This file stores the wo specific properties
#Mon Nov 18 16:07:32CET 2002
wo.wolocalroot=C:\\Apple\\Local
wo.woroot=C:\\Apple

Take note of the double back-slashes “\\”.
Add Local Framework doesn't work. You get three versions  
Importing projects with java code organised in packages doesn't work properly. Either the java files are not found or they may be put in the wrong directory. Adding them manually is not a big task!
