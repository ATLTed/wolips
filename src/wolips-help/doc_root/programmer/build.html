Building WOLips from source

  1. Install Ant on your system
  2. Download the latest WOProject source from SourceForge
  3. Change to the WOProject root directory
  4. Execute ant (dist is the default target in build.xml, and that's what you want)
  5. You'll find the result in the dist directory. WOLips_x.y.z corresponds to what you get when you download prebuilt versions of WOLips.
