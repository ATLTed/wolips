Working with WOLips in the PDE

  1. Check out the stuff at woproject/src/wolips
  2. Add cayenne.jar, log4j-1.2beta4.jar and woproject.jar from the woproject distribution to the wolips/lib folder.
  3. Go to preferences->java->classpath variables and add a variable JAVA_WO_FOUNDATION add let it point the foundation.jar.

You should now be able to run it as a Runtime Workbench.
